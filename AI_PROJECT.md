# AI Project Rules

This file contains project-specific AI rules for the Utils library.

This is a Java utility library providing unit conversion utilities (BitUnit, ByteUnit).

## Technology Stack

- Language: Java
- Build Tool: Maven
- Testing: JUnit

## Project-Specific Guidelines

- Follow existing code conventions in the project
- Ensure all public methods have proper Javadoc documentation
- Write unit tests for new functionality
